Responsive Table

This Demo shows how to make tables responsive for different screen resolutions without hide columns using the ResizeObserver and CSS.

  • assume each column width is 10rem
  • floated columns at end
  • observes the size of the container element and add css classes
  • javascript needed to take the width of the container through ResizeObserver
  • use html table
